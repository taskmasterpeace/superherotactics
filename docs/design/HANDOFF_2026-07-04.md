## The cost model (with the E/W win)

- Token = 3 gens. **Template animation** (49 presets) = **1 gen/direction**. **Custom animation** (gun-fire/beam/flight — no preset) = **~1 gen per FRAME per direction (~8×)**.
- **The E/W + mirror + cone facing model** collapses this to ~1 generated direction per animation: a Full-P0 character drops from **~65 → ~18 gens** (~72% cheaper). A 30-hero roster ≈ **~420–540 gens** — a fraction of the ~2,000/month plan.
- **Money is not the constraint** — generation *time* (batchable) + engine wiring are. Cheap wins: mirror E/W, lean on templates, pose+procedural FX for muzzle/recoil. Full cutout-rigging = overkill.
