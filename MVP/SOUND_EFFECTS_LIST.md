## IMPLEMENTATION NOTES

### Priority Tiers

**Tier 1 - Essential (Ship Blockers):**
- All weapon fire sounds
- Hit/miss feedback
- Death sounds
- Turn indicators
- Power activation sounds

**Tier 2 - Important (Polish):**
- All martial arts techniques
- Status effect sounds
- Cover system sounds
- Environmental destruction

**Tier 3 - Nice to Have:**
- Ambient backgrounds
- Detailed footsteps
- Menu UI sounds
- Victory fanfares

### Audio Format
- Format: OGG Vorbis (web-compatible, good compression)
- Fallback: MP3
- Sample Rate: 44.1kHz
- Bit Depth: 16-bit
- Channels: Mono (positional) or Stereo (UI)

### Volume Levels
- Sound Effects: -6dB baseline
- UI Sounds: -12dB (subtle)
- Combat Hits: -3dB (impactful)
- Ambient: -18dB (background)

### Positional Audio
- Use Web Audio API for 3D positioning
- Combat sounds positioned at unit location
- UI sounds non-positional (centered)
