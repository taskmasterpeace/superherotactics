## Notes on Name Selection

- Names were selected to be culturally appropriate for each country
- Mixed naming conventions were used where appropriate (e.g., French/local for former French colonies)
- Royal or dynastic family names were used for Gulf states (Al-Sabah, Al-Thani, Al-Maktoum)
- Common surname patterns were followed (e.g., -yan for Armenia, -ov for Central Asia, -ski for Poland)
- Some names reference real historical or political figures to add authenticity
- All names are fictional and not meant to represent any real individuals
